# Orologio della Piazza – PWA

Webapp autonoma e installabile, senza backend e senza database.

## Pubblicazione
Caricare l'intero contenuto della cartella su un hosting HTTPS, per esempio GitHub Pages, Netlify, Cloudflare Pages o un normale spazio web.

## Uso
1. Aprire la webapp dal browser.
2. Premere “Attiva campane”, necessario per autorizzare l'audio.
3. Su Android/Chrome usare “Installa l’app” oppure “Aggiungi a schermata Home”.

## Limite tecnico
I browser mobili non garantiscono la riproduzione audio quando la pagina è chiusa, sospesa o terminata dal sistema. La versione Android nativa rimane quindi la soluzione corretta per rintocchi affidabili in background.
